\doxysubsection{Detailed Description}
This class models a 3-\/D\+OF (Degree of Freedom) R\+RR (Revolute-\/\+Revolute-\/\+Revolute) type robotic arm. 

It provides functionalities to set and get joint angles, link lengths, and the end-\/effector position. Additionally, it offers methods to solve forward and inverse kinematics in multiple ways.
